# Investment Behaviour data-Analysis (Interactive Dashboard creation using power BI)
## Project Objective
"The objective of this Power BI project is to analyze and visualize investor behaviour to identify key trends in assests preference and risk tolerance . By leveraging data-driven insights, the project aims to help financial stakeholders understand the factors influencing investment decisions and optimize portfolio strategies for different demographic segments."
